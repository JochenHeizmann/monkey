
#Module opengl.databuffer


#Class DataBuffer

Data buffers are used by the following opengl commands: glVertexPointer, glColorPointer, glNormalPointer, glTexCoordPointer, glDrawElements, glBufferData, glBufferSubData, glTexImage2D, glTexSubImage2D, glCompressedTexImage2D, glCompressedTexSubImage2D, glReadPixels

#Method Size

Syntax::

Method @Size()

Description::

Returns the size, in bytes, of the data buffer.


#Method PeekByte

Syntax::

Method @PeekByte( @addr )

Parameters::

@addr - byte address to peek

Description::

Returns the byte value contained in the data buffer at address @addr.


#Method PeekShort

Syntax::

Method @PeekShort( @addr )

Parameters::

Description::
