Function JoyDown

Check if a joystick button is down.

Module: mojo.input

Syntax

Function JoyDown:Int( button:Int )

Parameters

button - joystick button to test.

Description

Return 1 if the specified joystick button is currently held down, else 0.

The button parameter should be one of the following constants:

JOY_A
JOY_B
JOY_X
JOY_Y
JOY_LB
JOY_RB
JOY_BACK
JOY_START
JOY_LEFT
JOY_UP
JOY_RIGHT
JOY_DOWN
