Function ATan2r

Compute arc tangent.

Module: mojo.math

Syntax

Function ATan2r:Float( x:Float, y:Float )

Parameters

x, y - floating point values.

Description

Returns the arc tangent of x / y, expressed in radians.

The function uses the signs of x and y to compute the correct quadrant for the result.
