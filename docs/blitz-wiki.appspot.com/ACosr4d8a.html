Function ACosr

Compute arc cosine of a value.

Module:: monkey.math

Syntax

Function ACosr:Float( x:Float )

Parameters

x - floating point value in the range -1 to 1 (inclusive).

Description

Returns the arc cosine of x, expressed in radians.
