Function Clamp

Clamp value to range.

Module: monkey.math

Syntax

Function Clamp( x, min, max )

Function Clamp:Float( x:Float, min:Float, max:Float )

Parameters

x - value to clamp

min - minimum returnable value

max - maximum returnable value

Description

Clamps x to the range min through max inclusive.
