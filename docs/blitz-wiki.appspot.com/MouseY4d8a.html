Function MouseY

Get y coordinate of mouse.

Module: mojo.input

Syntax

Function MouseY:Float()

Description

Returns the y coordinate of the mouse pointer.

On devices with a touch screen but no mouse, MouseY will instead return TouchY( 0 ).
